# Codenames Generator

status: done · ts, html, css

i was on a roadtrip and we tried to play codenames. it was impossible to balance a board on my lap, intuitively, i looked for codenames generator. coudlnt find a single one, somehow they only generate either the grid without words or the secret card.

after the trip and 15MB later:

[https://codenames-generator-bay.vercel.app](https://codenames-generator-bay.vercel.app)

it can generate an actual board with category of words, kids friendly included

it can be shared via qr code, including 2 views: agent and spymaster with password

especially it is completely offline friendly, you just need to load the web once, it will just work.

everything works without internet, yes the sharing works too!
